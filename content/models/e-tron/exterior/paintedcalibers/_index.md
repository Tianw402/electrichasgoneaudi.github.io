The code is **PC2** on e-tron 55/50 and **PC4** on e-tron S.

![Painted calibers](paintedcalibers.png "Painted Calibers on e-tron S")

The color code in different formats are.

- HEX f36f21
- RGB 243 111 33
- Lab 63 49 64
- CMYK 0 70 98 0
- HSB 22 86 95
- NCS S1080 Y50R
